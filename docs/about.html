About this website

This website is an IS 117 assignment for NJIT professor Keith Williams. Each of the chapters is writing done by me to summarize topics that I have learned about.

Assignment Details

For this website using Bootstrap or raw HTML/CSS you need to work individually to combine your how the Internet works site content with the history of the Internet content to create the most engaging and content rich site that you can, which demonstrates your personal website development ability and knowledge of the Internet. You should use a design program like Figma to create a visual mockup of the site and then implement this mockup in the actual coded site. You need to link the PDF of your Design in a ReadMe that appears on the project page of the GitHub repo.

Basic Requirements - 2 Points Each

  1. Home Page with a Z-Layout, Hero Text, Hero Image, and a call to action and a footer on all the pages
  2. Favicon - Click Here for more info
  3. An F-Layout for content pages
  4. SEO On Page Optimization - title tag, heading tags, Meta tags content, image alt tags and generally following this guide: SEO Checklist
  5. Google Analytics Tracking

Qualitative Requirements - 30 points each

  1. Visual Design - Is the site visually appealing and looks professional? You must design the site using a graphics tool like Figma, Axure, Photoshop, or any other tool that allows you to lay the site out before development
  2. Content - Does the site have authentic content that is presented in a way that communicates the objective of the site to the intended audience?

At least 5 Content Pages - 10 points each

Submission Requirements - 10 points following submission requirements

  1. A link to the site hosted on GitHub Pages
  2. A link to the GitHub repo of the site with links to the design and before/after PDF files
  3. ReadMe file with a description of the project and a link to your design as a PDF or individual page images
